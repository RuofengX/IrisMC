## 渠道
当前共三个分发渠道  

### Edge  
    Edge为经常更新的离线版本，有较新的特性，当然也可能有未被测试出的Bug  

### Edge-live
    Edge的在线更新版本，在HMCL启动时需要联网更新才可启动，并且依赖服务器状态。好处是更新频率最高。  

### Table
    经过较为完善的测试的Edge稳定版，更新频率很低。
